# Open Graph Images & Resources

This release adds comprehensive Open Graph image support and enhances the resource management system.

## Open Graph Images

- **Dynamic OG Generation**: Dynamic icon generation endpoint with customizable options
- **OG Image Layout**: Adjusted OpenGraph image layout and styling
- **Page OG Images**: Added Open Graph images for various pages
- **Metadata Standardization**: Standardized generateMetadata functions with explicit return types

## Resource Management

- **Resource URL Utilities**: Added resource URL utility functions and improved URL handling
- **External Config**: Added external resource configuration module
- **Header Sections**: Updated resource URL mapping and header section rendering
- **Empty State**: Hide sections when resource data is empty
- **Bilingual Support**: Removed bilingual versions for cleaner resource management

## Project Updates

- **New Projects**: Added new project page for Buddy Sensei and ONG website
- **Project Metadata**: Updated project metadata and added new SaaS category
- **Tags & Roles**: Updated project tags and roles for clarity
- **Cleanup**: Removed outdated project pages and files

## Technical Improvements

- **Slug Parameters**: Added slug parameter to multiple components and hooks
- **Locale Fallback**: Fixed locale fallback and improved image layout
- **Bot Protection**: Enhanced bot protection with reusable Turnstile component
- **Effect Dependencies**: Updated effect dependencies to include locale

This release significantly improves the portfolio's social media presence and resource management capabilities.
