# Subscription System & Thoughts Integration

This release introduces a subscription modal with Brevo integration and integrates a Thoughts section into the homepage.

## New Features

### Subscription Modal

- **Multi-Step Flow**: Revamped SubscriptionModal with multi-step flow
- **Brevo Integration**: Brevo API integration for email management
- **Auto-Subscribe**: Auto-subscribe on mount when email prop is provided
- **Product Validation**: Email API with product validation

### Thoughts Section

- **Homepage Integration**: Thoughts section integrated into Home page
- **New Component**: New Thoughts section component
- **Testimonials**: Enhanced testimonials with auto-rotating card design

### Technical Improvements

- **RPC Calls**: Improved RPC calls for user and subscriber handling
- **User Linking**: User linking and merging support in backend
- **Database Triggers**: Trigger inspection SQL for newsletter subscribers
- **Unique Constraints**: Prevent duplicate emails and user IDs

This release enhances user engagement with subscription capabilities and improved content presentation.
