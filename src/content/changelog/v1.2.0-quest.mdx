# Quests & Talks System

This release introduces a comprehensive quests and talks system with full internationalization support.

## New Features

### Quests System

- **Quest Pages**: Complete quest pages with registration and submission flows
- **Quest Cards**: New QuestCard UI with emoji covers
- **My Quests Section**: Card-based My Quests section
- **Registration Flow**: Streamlined quest enrollment and submission forms
- **Winners Support**: Quest winners support and UI
- **Rewards UI**: Rewards UI for quests
- **Feedback Modal**: Quest feedback modal and mail templates

### Talks System

- **Talks Page**: New Talks page UI
- **Participants**: Talk participants avatars and stats
- **Attendance Mode**: Attendance mode support for talks
- **Event URLs**: Event/replay URLs and actions for talks
- **External URLs**: Enable external talk URLs and previews

### Internationalization

- **Comprehensive i18n**: Full i18n integration for quests and talks
- **Translations**: French and Chinese translations
- **Locale Metadata**: Locale-aware metadata and redirects
- **Multi-language**: Support for multiple languages across all features

## Technical Improvements

- **Content Collections**: Refactor content to use collections; add APIs
- **Icon Generator**: New icon generator
- **Open Graph**: Remove dynamic Open Graph image endpoint
- **ESLint**: Added ESLint config
- **MCP Setup**: MCP server setup
- **Proxy Refactor**: Proxy refactor for better performance

This release adds powerful new features for managing quests and talks with full internationalization support.
