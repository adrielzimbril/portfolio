# Mur communautaire & interface pilotée par les données

Cette version introduit un mur communautaire avec un design en cartes, des composants de tableau de bord pilotés par les données et une nouvelle page de changelog.

## Mur communautaire

### Design en cartes

- **Cartes à motifs SVG** : mur communautaire visuel avec des cartes à motifs SVG
- **Rendu des avatars** : meilleur rendu des avatars avec couleurs de secours
- **Laisser une note** : les utilisateurs peuvent laisser des notes et des messages
- **Modale de filtre** : modale de timeline pour filtrer les messages

### Authentification

- **Intégration Supabase** : base de données temps réel pour les mises à jour instantanées
- **OAuth GitHub** : connexion avec un compte GitHub
- **Gestion utilisateur** : gestion complète des profils utilisateurs

## Interface pilotée par les données

### Tableau de bord des statistiques

- **Composants data-driven** : migration des statistiques vers des composants alimentés par les données
- **Stats GitHub** : graphes de contribution et métriques GitHub
- **Scores Lighthouse** : suivi des performances avec durée de cache
- **Vues de pages** : suivi analytics avec mises à jour atomiques

### Outils & setup

- **Cartes data-driven** : migration des outils et du setup vers des cartes alimentées par les données
- **Migration des icônes** : migration des icônes vers des SVG pour de meilleures performances
- **Gestion des ressources** : gestion centralisée des ressources

## Page changelog

### Nouvelles fonctionnalités

- **Interface du changelog** : nouvelle page changelog avec en-tête et timeline
- **Widget de mises à jour** : widget animé des dernières mises à jour
- **Couche de données** : couche de données pour récupérer les releases
- **Support des réactions** : réactions sur les entrées du changelog

## Améliorations techniques

- **Bibliothèque d'icônes** : migration vers la nouvelle bibliothèque aurthle/icons
- **Motion UI** : adoption de Motion UI pour des animations plus fluides
- **Tokens de couleur** : migration des couleurs UI vers les tokens squircle
- **I18n** : amélioration du support international
- **Logger** : ajout d'un logger pour remplacer les erreurs console

Cette version améliore fortement l'expérience utilisateur grâce à un mur communautaire plus soigné et des composants modernes pilotés par les données.
