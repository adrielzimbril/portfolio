# Projects Gallery & Theme System

This release introduces a projects gallery and theme switching capabilities.

## New Features

### Projects Gallery

- **Filtering System**: Filter projects by category and tags
- **Grid Layout**: Responsive grid layout for project cards
- **Project Details**: Detailed project pages with case studies
- **Search Functionality**: Search projects by keywords

### Theme Switcher

- **Light/Dark Mode**: Toggle between light and dark themes
- **System Preference**: Respect system theme preference
- **Theme Persistence**: Remember theme choice across sessions
- **Smooth Transitions**: Smooth theme transition animations

### Footer Enhancements

- **Social Links**: Links to social media profiles
- **Contact Info**: Updated contact information
- **Quick Links**: Quick navigation to important sections
- **Copyright**: Copyright information and licensing

These features enhance the portfolio's visual appeal and user experience.
