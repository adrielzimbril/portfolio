# Page View Tracking & Analytics

This release introduces comprehensive page view tracking and analytics capabilities to the portfolio.

## New Features

### Page View Tracking

- **Server-Side Tracking**: Robust server-side analytics tracking with atomic updates
- **IP Info Fetching**: Enhanced IP lookup handling with validation and caching
- **View Badges**: ViewsBadge and ViewsIncrementor components for displaying view counts
- **Multi-Page Support**: Page view tracking integrated into multiple content pages

### Analytics API

- **Page Views Endpoint**: API endpoint for tracking and retrieving page views
- **Unique Views**: Functions for handling unique view upserts with proper validation
- **RPC Calls**: Switched to RPC for efficient analytics data retrieval
- **Database Schema**: Supabase page_views schema with proper constraints

### Technical Improvements

- **Atomic Updates**: Server-side atomic updates for accurate view counting
- **Caching**: Enhanced caching for IP lookup and analytics data
- **Validation**: Robust validation for page view data handling
- **Integration**: Seamless integration into content pages

This release provides valuable insights into user engagement and content performance across the portfolio.
