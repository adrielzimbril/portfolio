# 🌍 Internationalization (i18n)

The project uses `next-intl` to provide a full multilingual experience (English, French, and Chinese).

## 📁 File Locations

- **Translation JSON files**: `src/integrations/i18n/translations/`
    - `fr.json` (Source/Reference)
    - `en.json`
    - `zh-CN.json`
- **Personal Data**: `src/data/personal/translate/`
- **MDX Content**: Localized files use the format `[filename].[locale].mdx` (e.g., `about.en.mdx`) in `src/content/`.

## 🤖 Automated Translation (Lobe i18n)

The entire project is translated automatically using **[Lobe i18n](https://github.com/lobehub/lobe-cli-i18n)**.

### How to Translate
To run the automated translation, use the following command:
```bash
pnpm translate
```

### Configuration
You can customize the translation behavior in the **`.i18nrc.ts`** file at the root of the project.

#### Translation Modes
In `.i18nrc.ts`, you can switch between two modes by modifying the `config` object:

```typescript
const config = {
  modelName: "gpt-5-nano",
  mode: "default", // "default" for JSON translation files | "content" for personal data
  isMdx: false,    // Set to true to translate MDX files | false for standard MD files
};
```

- **`mode: "default"`**: Translates the main interface strings in `src/integrations/i18n/translations/`.
- **`mode: "content"`**: Translates the data in `src/data/personal/translate/`.
- **`isMdx: true`**: Targets all `.mdx` files in the `src/content/` directory.

## 🛠️ Usage in Code

### Client Components
```tsx
import { useTranslations } from 'next-intl';

const t = useTranslations('home');
return <h1>{t('title')}</h1>;
```

### Server Components
```tsx
import { getTranslations } from 'next-intl/server';

const t = await getTranslations('home');
return <h1>{t('title')}</h1>;
```

## ➕ Adding New Translations

1. Add the key and its value in **`fr.json`** first.
2. Run `pnpm translate` (in `default` mode) to automatically generate the English and Chinese versions.
3. Use the key in your component using the `useTranslations` or `getTranslations` hook.
