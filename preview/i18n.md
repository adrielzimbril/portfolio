# 🌍 Internationalization (i18n)

The project uses `next-intl` to provide a full multilingual experience (English, French, and Chinese).

## 📁 File Locations

- **Translation JSON files**: `src/integrations/i18n/translations/`
    - `fr.json` (Source/Reference)
    - `en.json`
    - `zh-CN.json`
- **MDX Content**: Localized files use the format `[filename].[locale].mdx` (e.g., `about.en.mdx`).

## 🛠️ Usage in Code

### Client Components
```tsx
import { useTranslations } from 'next-intl';

const t = useTranslations('home');
return <h1>{t('title')}</h1>;
```

### Server Components
```tsx
import { getTranslations } from 'next-intl/server';

const t = await getTranslations('home');
return <h1>{t('title')}</h1>;
```

## ➕ Adding New Translations

1. Add the key and its value in **`fr.json`** first.
2. Provide the corresponding translations in **`en.json`** and **`zh-CN.json`**.
3. Use the key in your component using the `useTranslations` or `getTranslations` hook.
